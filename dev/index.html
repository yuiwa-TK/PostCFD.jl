PostCFD.read_dividedgrid — Method

read divided grid files. (eg. grid files in dir of "/Grid/Work.divide/comp/" in HPC Template. ) This function returns xyz_vector=Vector{Array{Float64}(jmax,kmax,lmax,3)}. Arg1 : name of directory that contains the divided grid files. Arg2 : grid rank(number) indicated by range (eg. 1:2:19) or array (e.g. [1,3,5,⋯,19]) where you will read.
